📌 Modelagem de Fluxos de Caminhos – Tela de Login
📖 Sobre o Projeto

Este documento descreve a modelagem de fluxos de interação de uma tela de login, utilizando o conceito de grafos direcionados. A análise foi desenvolvida no contexto do curso de Análise e Desenvolvimento de Sistemas (3º período).

O objetivo é representar visualmente e logicamente os possíveis caminhos que um usuário pode seguir ao interagir com a interface.
